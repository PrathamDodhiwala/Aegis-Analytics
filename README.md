# 🛡️ Aegis Analytics // Enterprise Fraud Console

---

# 📌 Overview

This project simulates a real-world banking fraud monitoring system capable of:

- Detecting suspicious transactions
- Predicting fraud probability
- Monitoring live transaction analytics
- Generating downloadable fraud reports
- Visualizing business intelligence insights

The application uses a Random Forest Machine Learning model trained on transaction behavior data.
